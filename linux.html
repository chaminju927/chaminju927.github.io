#3. Linux 명령어 정리

    1. 파일시스템 탐색 기본 명령어

  • pwd - 현재 작업 디렉토리를 표시
    ex) $pwd
    /home/vagrant
  • cd - 특정 디렉토리로 이동. cd [디렉토리명]
    ex) [vagrant@host1 ~]$ cd git
    [vagrant@host1 git]$
  • ls - 현재 위치의 디렉토리 내용 리스트 출력.
    ex)[vagrant@host1 git]$ ls
    bitcamp-ncp bitcamp-ncp2 bitcamp-study bitcamp-test
  • file - 파일 종류 확인할때 사용. file [파일명]
    ex)$ file c.txt
    c. txt: ASCll text
  • less - 파일 내용을 한 화면에 보여주는 명령어, 파일 내용을 보여주는 화면에서 나갈때는 q

    2. 파일과 디렉토리 조작 명령어

  • cp - 기존 파일을 복사해 새 파일을 생성한다. cp [기존파일명] [새파일명]
    ex) $ cp file1 file2
    $ ls
    file1 file2
  • mv - 파일 위치를 이동하는 명령어. 같은 폴더에서 파일, 디렉토리 이동을 하는 경우 이름 변경도 가능.
    - 파일명 변경하는 경우 mv [기존파일명] [새파일명]
    - 파일 위치를 이동하는 경우 mv [이동할파일명]
  • rm - remove의 약자로 디렉토리를 삭제할때 사용하는 명령어.
    ex) ~ git]$ rm -rf 11
    ~ git]$ ls
    bitcamp-ncp bitcamp-ncp2 bitcamp-study bitcamp-test
  • ln - link의 약자로 링크파일을 만들때 사용한다. 원본이 삭제되어도 사용가능한 하드링크와,
    원본이 삭제돼 사용할 수 없는 심볼릭링크가 있다.
    - 하드링크 : $ ln [원본링크] [링크이름]
    - 심볼릭링크 : $ ln -s [원본링크] [링크이름]

    3. 명령어를 다루는 명령어

  • type - 파일 내용을 하나로 합칠 때 사용한다.
    ex) $ type 1.txt 2.txt 3.txt > 4.txt
  • which - 실행 파일의 위치를 알 수 있다.
    ex) $ which unzip
    /usr/bin/unzip
  • man - 각종 명령어나 프로그램의 매뉴얼을 확인하는 명령어이다.
    ex) $ man rm
    RM(1)
    NAME
    rm - remove files of directories
    SYNOPSIS
    rm [OPTION]... FILE...
    DESCRIPTION
    ...
  • apropos - 검색어와 관련 있는 명령어를 설명과 함께 출력하는 명령어이다.
    ex) $ apropos rm
    sysinfo (2) - returns information on overall system statistics
    Config (3pm) - access Perl configuration information
    Pod::Perldoc (3pm) - Look up Perl documentation in Pod format.
    ...
  • info - 명령어의 사용 방법이나 옵션 등을 나타낸다.
    ex)$ info rm
    File: coreutils.info, Node: rm invocation, Next: shred invocation, Prev: mv invocation, Up: Basic operations
    11.5 'rm': Remove files or directories
    ======================================
    'rm' removes each given FILE. By default, it does not remove
    directories. Synopsis:
    rm [OPTION]... [FILE]...
    ....
  • whatis - 명령어에 대한 기능을 간략하게 나타낸다.
    ex) $ whatis rm
    rm (1) - remove files or directories
    rm (1p) - remove directory entries
    .....
  • alias - 명령어를 간소화해 다른이름으로 사용할 수 있게 해주는 명령어이다.
    ex) $ alias minju = cd git/bitcamp
    $ minju
    ~ git bitcamp]$
