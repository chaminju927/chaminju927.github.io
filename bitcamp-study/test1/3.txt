3. 명령어를 다루는 명령어

type - 파일 내용을 하나로 합칠 때 사용한다.
 $ type 1.txt 2.txt 3.txt > 4.txt

 
which - 실행 파일의 위치를 알 수 있다.
 $ which unzip
  /usr/bin/unzip


man - 각종 명령어나 프로그램의 매뉴얼을 확인하는 명령어이다.
 $ man rm
RM(1)

NAME
        rm - remove files of directories
SYNOPSIS
        rm [OPTION]... FILE...
DESCRIPTION 
...



apropos - 검색어와 관련 있는 명령어를 설명과 함께 출력하는 명령어.
 $ apropos rm
sysinfo (2)          - returns information on overall system statistics
Config (3pm)         - access Perl configuration information
Pod::Perldoc (3pm)   - Look up Perl documentation in Pod format.
...



info - 명령어의 사용 방법이나 옵션 등을 나타냄.
 $ info rm
File: coreutils.info,  Node: rm invocation,  Next: shred invocation,  Prev: mv invocation,  Up: Basic operations

11.5 'rm': Remove files or directories
======================================

'rm' removes each given FILE.  By default, it does not remove
directories.  Synopsis:

     rm [OPTION]... [FILE]...
....



whatis - 명령어에 대한 기능을 간략하게 나타냄.
$ whatis rm
rm (1)               - remove files or directories
rm (1p)              - remove directory entries
.....



alias -  명령어를 간소화해 다른이름으로 사용할 수 있게 해주는 명령어.
 $ alias minju = cd git/bitcamp
 $ minju
 ~ git bitcamp]$
