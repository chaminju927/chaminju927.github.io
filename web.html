#4. Web기술의 등장

어느 단체의 누가 어떤 목적으로 웹 기술을 만들었나?

영국의 컴퓨터 공학자 팀 버너스 리에 의해 1989년 월드 와이드 웹의 하이퍼 텍스트 시스템이 개발되었다.
최초의 URL, HTML , HTTP 설계와 구현은 유럽 입자 물리 연구소에서 그와 동료였던 로베르 카이오 등과 함께
만들어진 것이며, 개발의 본래 목적은 물리학자들의 문헌 검색 및 제휴를 위함이었다.

웹 기술의 처음 목적과 지금 활용되는 상황이 어떻게 다른가?

웹 기술의 처음 목적은 문서를 읽는 기능을 갖는 것이었다. 웹1.0은 주로 회사나 개인에 의해 운영되며
오로지 ‘읽기’만 가능했고, 클릭 외에는 어떠한 상호 작용도 없는 정적인 웹페이지였다.

하지만 2000년대에 들어서면서, 인터넷 보급 확산과 디바이스 다양화로 웹은 급속하게 발전했고 단순 '읽기'에서
직접 컨텐츠를 ‘생산’하고 ‘공유’하며 적극적으로 ‘참여’하기 시작했다. 웹 1.0은 단순한 웹페이지들의 집합으로
단방향 커뮤니케이션만 가능했지만, 웹이 플랫폼의 형태로 발전한 웹 2.0은 양방향 커뮤니케이션이 가능해졌다.
이때는 이전과 달리 거대 플랫폼이 등장하여 웹 서비스를 운영하는 주체가 되었다.

더 나아가 웹 3.0은 1998년 팀 버너스 리가 제안한 개념인 ‘시맨틱 웹(Semantic Web)’이 그 시작이다.
시맨틱 웹은 '의미론적인 웹’을 뜻하는데 기계가 인간들이 사용하는 자연어를 이해하고 상황과 맥락에 맞는 개인 맞춤형
정보를 제공하는 웹을 의미한다. 지금의 웹 3.0은 시맨틱 기술을 활용해 개인 맞춤형 정보를 제공하고
블록체인 시스템을 통한 탈 중앙화와 데이터 암호화에 기반한 ‘개인의 데이터 소유’가 가능해진 새로운 형태의
웹 생태계를 의미한다고 할 수 있다. 웹 3.0의 대표적인 서비스는 암호자산·NFT·메타버스 등으로,
웹 3.0 세계에서는 플랫폼 사업자, 금융기관 등의 중개자를 거치지 않고 개인과 개인이 연결되어 거래가 가능하다.
즉, 인간이나 플랫폼에 의한 것이 아니라 네트워크를 통해 운영되는 것이라 볼 수 있다.
